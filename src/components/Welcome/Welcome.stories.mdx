# wood-ui 一款个性的ui框架
wood-ui 是一套功能丰富的组件库，使用 React Hooks 和 typescript
### 安装

~~~javascript
npm install wood-ui --save
~~~
### 使用
~~~javascript
// 加载样式
import 'wood-ui/dist/index.css'
// 引入组件
import { Button } from 'wood-ui'
~~~
